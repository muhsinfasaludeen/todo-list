# 📝 ToDo Web Application

## 📌 Overview
This is a simple and modern ToDo web application developed as part of the technical evaluation for the AI / Software Development Intern position at iLogicPlus Solutions LLP.

The application allows users to:
- Add tasks
- Mark tasks as completed
- Delete tasks
- View total number of tasks
- Persist tasks using localStorage (data remains after refresh)

---

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript
- Browser localStorage API

---

## ⚙️ Features
- Add, complete, and delete tasks
- Real-time DOM updates
- Persistent data storage using localStorage
- Modern and clean UI with gradient background and card layout
- Responsive and user-friendly design

---

## ✨ Enhancements
- Implemented **localStorage** to store tasks permanently in the browser
- Designed a **modern UI** using gradients, shadows, and card-based layout
- Improved user experience with interactive task elements
